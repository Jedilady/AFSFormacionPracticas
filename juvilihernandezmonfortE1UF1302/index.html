Caso práctico

Para crear un documento HTML sencillo, utilizando VSCode, crea una carpeta nueva, y dentro un documento index.html. Escribe la estructura básica del HTML, comenzando con !DOCTYPE html (con la ayuda de VSCode, selecciona html:5) Dentro de head, dale un título a tu página bajo la etiqueta title. En el body, añade el contenido que quieres mostrar. Recuerda dividiro bajo las etiquetas header para la cabecera (donde podrías colocar el bloque nav para el menú y un h1 para el título principal), main para especificar tu contenido, y footer para colocar un pie de página con información.

  • Etiquetas (Tags): Elementos básicos en HTML que definen el contenido y la estructura de una página
  • Atributos: Características adicionales que se pueden agregar a las etiquetas para proporcionar información extra, como href
  • Elementos: Combinación de una etiqueta de apertura, el contenido, y una etiqueta de cierre
  • Estructura de Documento: La organización fundamental de un documento HTML
  • Encabezados (Headings)Etiquetas como h1, h2, h3, etc., que definen la jerarquía de los títulos en una página web.
  • Listas: Estructuras para organizar contenido en formato de lista
